# Beeswarm Plot — Employee Compensation

Email: 24f3000719@ds.study.iitm.ac.in

This repository contains a Beeswarm-style plot (chart.png) visualizing employee salary distribution across departments and experience levels.

Files:
- data.csv — sample dataset used to create the plot
- chart.png — raster export (400x400 px)

To reproduce in RAWGraphs:
1. Visit https://rawgraphs.io/
2. Import the contents of `data.csv` (copy-paste).
3. Select 'Beeswarm Plot' and map `salary` to X, `department` to Y (or Category), and `level` to Color.
4. Customize color palette and point size, then export as PNG (300–512 px).
